# Shop phụ kiện chó mèo


## Demo: https://nextjs-ecommerce-six.vercel.app/





## Cài đặt phần phụ thuộc
### `npm cài đặt`

## Kết nối với mongodb của bạn và thêm thông tin vào next.config.js

## Chạy dự án http: // localhost: 3000
### `npm run dev`
